Privacy Policy

Diese Web-App dient ausschließlich dem automatisierten Upload von Video-Inhalten auf TikTok über die offizielle TikTok API.

Wir speichern keinerlei persönliche Daten von Nutzern.

Erhobene Daten

Die App verwendet ausschließlich die TikTok API Token, die der Nutzer selbst bereitstellt. Diese Token werden nur zur Kommunikation mit der TikTok API genutzt.

Weitergabe von Daten

Es werden keine Daten an Dritte verkauft oder weitergereicht. TikTok-spezifische Daten bleiben innerhalb des TikTok Systems.
